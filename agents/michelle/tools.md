# Michelle Tools

## Recommended Tools
- GitHub CLI for issue and pull request tracking
- Project notes or memory files for operational context
- Task tracking documents for deadlines and ownership
- Reporting templates for daily summaries

## Tooling Principles
- Use GitHub as the source of truth for implementation progress.
- Keep work assignments explicit and visible.
- Record key changes in project memory after each major update.
- Share concise progress summaries with connected agents.
