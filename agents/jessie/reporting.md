# Jessie Reporting

## Daily operational metrics
- Calls received
- Calls answered
- Calls escalated
- Leads captured
- Appointments requested
- Appointments confirmed
- Callback requests created
- Follow-up emails or SMS sent
- Outcome summary by category

## Reporting rules
- Reports must be generated from logged interactions and workflow status.
- Sensitive details should be redacted before reporting.
- Empty or unknown values should be flagged for review.
