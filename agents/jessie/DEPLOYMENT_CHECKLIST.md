# Deployment checklist

Do not deploy until every item is complete.

- [ ] Use a non-production external sandbox first.
- [ ] Store credentials in a managed secret store.
- [ ] Add TLS, network controls, tenant isolation, and distributed rate limiting.
- [ ] Replace local JSON with an encrypted managed datastore.
- [ ] Verify consent, retention, deletion, access, and incident-response policies.
- [ ] Run call, scheduling, handoff, crisis-escalation, and redaction acceptance tests.
- [ ] Add monitoring, alerts, backups, recovery exercises, and token rotation.
- [ ] Obtain operational, privacy, security, and clinical-safety approval.
- [ ] Complete a staged launch with rollback controls.
