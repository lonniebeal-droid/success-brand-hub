# Agents

This directory contains all AI agent definitions, documentation, and configuration for the Success Brand Hub platform.

Purpose
- Centralize agent documentation and responsibilities so contributors and automation can discover and use agents consistently.

Contents
- One subdirectory per agent (for example: ceo, content, operations, sales, research, etc.).
- Each agent directory should include a README.md describing the agent's purpose, responsibilities, inputs, outputs, connected agents, future features, and current status.

Guidelines
- Keep each agent's README focused and actionable.
- Use the following README structure for every agent:
  - # Agent Name
  - ## Purpose
  - ## Responsibilities
  - ## Inputs
  - ## Outputs
  - ## Connected Agents
  - ## Future Features
  - ## Current Status

Status
- Planned
